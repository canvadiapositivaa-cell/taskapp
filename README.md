# TaskApp - Aplicación móvil para gestión de tareas

## Descripción
TaskApp es una aplicación móvil básica desarrollada con Ionic, Angular y Capacitor. Presenta una interfaz orientada a la gestión de tareas, con campo de entrada, botón de acción y listado de actividades pendientes.

## Objetivo
Desarrollar una aplicación móvil básica para la gestión de tareas, aplicando herramientas de desarrollo móvil multiplataforma y generando una versión instalable para Android.

## Tecnologías utilizadas
- Ionic Framework
- Angular
- Capacitor
- Node.js
- npm
- Android Studio
- GitHub

## Instalación y ejecución
Para instalar las dependencias:

npm install

Para ejecutar el proyecto:

ionic serve

Para construir el proyecto:

ionic build

Para sincronizar con Android:

npx cap sync android

Para abrir en Android Studio:

npx cap open android

## Evidencias
El proyecto incluye evidencias de instalación, creación del proyecto, integración de Capacitor, ejecución en servidor local, compilación en Android Studio, generación del APK e interfaz de gestión de tareas.
